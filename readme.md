# FSPM METAR VIEWER

## How to build the project
First you need to install the fspm (package manager)

```
> npm install -g fspm
```
To debug it run
```
> npm start 
```
And you shound see the app running in localhost
To build the app run
```
> npm run build 
```
It makes an app package in `./fspm-build/vfrmap.tar.gz`
